Parameterized Queries

A parameterized query is one in which there are placeholders that are filled in at runtime with values. SADL supports parameterized queries only in SPARQL query strings, not in SADL queries. For example,

    Ask ChildrenQuery: "select ?child where {? <child> ?child}" : [Wanda].

is a named query with a single parameter, which is the subject of the first triple in the where patterns. This query, once defined, can be used as follows:

   Ask: ChildrenQuery : [Mary].
    Ask: ChildrenQuery : [George].

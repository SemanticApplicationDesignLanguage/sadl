How to Check Java Version

Last revised 04/27/2020. Send questions and comments to A. W. Crapo.

Windows

  1. Click the Start button.

  2. Scroll through the applications and programs listed until you see the Java folder.

  3. Click on the Java folder, then About Java to see the Java version.

Mac

  1. Click on Apple icon on upper left of screen.
  2. Go to System Preferences
  3. Click on the Java icon to access the Java Control Panel.
  4. Click on About Java to see the Java version.

Linux

  1. Open a Linux Terminal (command prompt) window
  2. Enter the command "java -version".

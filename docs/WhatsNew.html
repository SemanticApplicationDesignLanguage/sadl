What's New in SADL

Last revised 06/09/2023. Contact us.
  1. (June 2023) UnittedQuantity support has been enhanced and reasoning with UnittedQuantity property ranges is supported by both the Jena-based reasoner and the SWI-Prolog reasoner. See Unitted Quantities in SADL and SWI-Prolog Reasoner/Translator Plug-in Pair for SADL.
  2. (Feb 2022) Wrap any Java class method that computes over primitive data types (int, float, string, etc.) in a SADL External equation to make the functionality available for use in Jena Rule's. See Wrapping Java Class Methods as Jena Built-ins for the Jena Reasoner.
  3. (May 2021) Reason with default values and/or stage rules for sequential application  in Jena Rules. See Default Values and Staged Rules in the SADL Jena Augmented Reasoner/Translator.

Inference

Last revised 6/4/2021. Contact us.

Inference is the process of deriving new statements, in the form of RDF Triples, from existing statements. In SADL models, inferences can be based on first order logic and set theory or it can be based on rules in the model. Drawing inferences of these two types must be integrated within the reasoner (the code that computes inferences) so that new statements from either source may be used to draw additional inferences of either type.

For a tutorial in inference in OWL see slides Part 3: The Web Ontology Language (OWL) or video Entailments, Rules, and Reasoning.

Importing Files into SADL

Last revised 09/17/2020.

The SADL-IDE allows importing two different types of files into a project and in some cases allows multiple ways of doing so and multiple final file formats. File types which can be imported include:

  • CSV files, which can be transformed into OWL models or SADL models using a translation template (see Tabular Data Importer for details)
  • OWL files, which can be translated into SADL models

To import either type of file, click on "File" on the Eclipse menu bar and select "Import...". In the resulting wizard, open the "SADL" option and select the type of file import desired. The wizard will prompt for additional information.

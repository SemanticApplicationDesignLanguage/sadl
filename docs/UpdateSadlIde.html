Instructions for Updating the Version 3 SADL-IDE

Last revised 10/26/2020. Contact us.

If you already have a recent release of SADL Version 3.4 installed, you can simply update instead of doing a fresh install. You should be running at least Eclipse 2020-03 and Java 8. If not, please get a fresh Eclipse and do a new installation, see Install Sadl IDE.

To update SADL Version 3 do the following:

  1. Download the latest SADL V3 installation ZIP file from https://github.com/SemanticApplicationDesignLanguage/sadl/releases.
  2. Start Eclipse, go to Help -> Install New Software. Click on Add... to create a new update site. Then click on Archive and browse to the ZIP file downloaded in step 1. Click OK to install SADL. Select the features. Because SADL is already installed, you will be told that requested install is being changed to update. Accept the license agreement, confirm installation and restart.

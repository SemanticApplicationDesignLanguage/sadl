Ad Hoc Queries in the SADL IDE

Last revised 4/13/2021. Contact us.

While queries can be persisted as part of a SADL model and run whenever desired, see Ask Statement,  it is also possible to run SPARQL queries against the current model in an ad hoc fashion. To do so, follow this sequence of actions.

  1. Identify the model to be queried. This can be done in either of two ways:
    • Open the SADL model in the SADL editor or the OWL model in a text editor.
    • Select the SADL or OWL file in the Project Explorer window.
  2. Open the ad hoc query dialog by selecting "Run Query" on the SADL menu. This can be made visible in either of two ways:
    • In the SADL perspective, click on "SADL" in the menu bar
    • Click on the SADL icon on the toolbar.
  3. Enter a SPARQL query into the resulting Run Query dialog.
  4. Check or uncheck "Show namespaces?", as desired
  5. Click the OK button.

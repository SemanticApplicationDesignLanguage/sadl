Topics in Semantic Modeling, SADL, and OWL: a Tutorial

Last revised 1/29/2021. Contact us.

Introduction

This document is the index to a sequence of topics in semantic modeling and in the use of the Semantic Application Design Language (SADL) to generate models expressed in the Web Ontology Language (OWL) plus rules. While the sequence is intended to provide a gentle introduction, an effort has also been made to allow each topic to be used independently by someone familiar with the technology in general and looking for specifics of how the SADL modeling approach may be applied to their particular problem. Later parts of the tutorial go into aspects of the SADL integrated development environment (IDE) useful for model validation, regression testing, maintenance, and knowledge base application for decision support.

Go to tutorial.

Approach

Believing that people learn best when they are engaged in inquiry and reflection, many of the slide decks and videos linked in the outline below pose problems and invite the user to pause and to work out his or her own solution to the exercise before continuing. One or more solutions to the problem are then presented. Users of this material are encouraged to engage in the learning process and to not take an entirely passive approach.

Engagement in solving problems and answering questions presupposes some mechanism of expression and model execution. This material uses the Semantic Application Design Language (SADL) IDE and language as means to this end. Therefore the user is invited to have SADL installed on their computer and to use it in solving the problems presented. See Installation Instructions for SADL-IDE, Version 3. It is also possible to use WebSADL for a browser-based IDE after installing the WebSADL server, see WebSadl installation at https://hub.docker.com/r/theiaide/sadl.

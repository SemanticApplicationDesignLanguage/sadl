Introduction

This document is the index to tutorials in semantic modeling and in the use of the Semantic Application Design Language (SADL) to generate models expressed in the Web Ontology Language (OWL) plus rules. While some topics depend on an understanding of other topics, suggesting a sequence, an effort has also been made to allow each topic to be used independently by someone familiar with the technology in general and looking for specifics of how the SADL modeling approach may be applied to their particular problem. Later tutorials go into aspects of the SADL integrated development environment (IDE) useful for model validation, regression testing, maintenance, and knowledge base application for decision support.

Approach

Believing that people learn best when they are engaged in inquiry and reflection, many of the slide decks and videos linked in the outline below pose problems and invite the user to pause and to work out his or her own solution to the exercise before continuing. One or more solutions to the problem are then presented. Users of this material are encouraged to engage in the learning process and to not take an entirely passive approach.

Engagement in solving problems and answering questions presupposes some mechanism of expression and model execution. Therefore, even for material that is not specifically about the Semantic Application Design Language (SADL) IDE and language, it is still used as means to allow concrete solution implementation and validation. Therefore the user is invited to have SADL installed on their computer and to use it in solving the problems presented. See Installation Instructions for SADL-IDE, Version 3. It is also possible to use WebSADL for a browser-based IDE after installing the WebSADL server, see WebSadl installation at https://hub.docker.com/r/theiaide/sadl.
